# **App Name**: Souq Lan

## Core Features:

- Multi-vendor Marketplace: Enable multiple vendors to list and sell products on the platform.
- Advanced Product Listing: Allow vendors to add products with multiple images, detailed descriptions, specifications, price, discount, condition, location, smart categorization, and tags.
- User Profiles and Management: Provide user registration/login (email, phone, Google), profile management, vendor following, product saving, and internal messaging.
- Vendor Dashboard: Equip vendors with a dashboard to add, modify, and delete products; view statistics; manage orders; access an earnings wallet; and verify their account.
- Admin Control Panel: An admin interface to manage users and vendors, products and categories, advertisements, reports, content review, and commission control.
- Internal Chat and Support: Enable direct communication between buyers and sellers, along with a support ticket system.
- Intelligent Recommendations: AI-powered system to analyze user behavior and product data to suggest relevant products, enhancing the shopping experience and increasing sales; utilizes a tool to identify item characteristics and compare them to customer profiles, to determine relevancy

## Style Guidelines:

- Use a dark color scheme, with a background of #1A1A1A, a primary color of #80CBC4 (a muted teal, aligning with a professional and modern look) and an analogous accent color of #80B3FF (a soft blue).
- Use 'Inter' (sans-serif) for both headlines and body text, lending a modern, neutral, and readable aesthetic.
- A modern, responsive layout that works seamlessly across mobile, tablet, and desktop devices.
- Font recommendation: Cairo / Inter / Poppins Note: currently only Google Fonts are supported.
- Implement micro-animations and smooth transitions to enhance user experience.
- Use modern, minimalist icons for clear navigation and visual appeal.
